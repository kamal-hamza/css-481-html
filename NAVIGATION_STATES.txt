NAVIGATION BUTTON STATES
========================

The navigation buttons (Home, Favorite Games, Book Collection) have
three distinct visual states:

1. DEFAULT STATE
   - Text Color: white (#ffffff)
   - Background: dark gray (#333333 / var(--secondary-color))
   - Border: 2px solid white

2. HOVER STATE
   - Text Color: black (#000000)
   - Background: white (#ffffff)
   - Border: 2px solid white
   - Effect: Slight lift (translateY) and scale
   - Shadow: Hover shadow appears

3. ACTIVE PAGE STATE
   - Text Color: black (#000000)
   - Background: white (#ffffff)
   - Border: 2px solid white
   - Indicates: Currently viewing this page

CSS IMPLEMENTATION
==================

Default:
header.site-header nav.navigation ul.nav-list li.nav-item a.nav-link {
    color: white;
    background-color: var(--secondary-color);
    border: 2px solid white;
}

Hover:
header.site-header nav.navigation ul.nav-list li.nav-item a.nav-link:hover {
    background-color: white;
    color: #000000;
}

Active:
header.site-header nav.navigation ul.nav-list li.nav-item a.nav-link.active {
    background-color: white;
    color: #000000;
}

==============
